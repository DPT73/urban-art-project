# Images Directory

## Structure des images requises

Placez vos images dans ce dossier selon la structure suivante :

### Photographie (6 images)
- `photo1.jpg` - Photo urbaine 1
- `photo2.jpg` - Photo urbaine 2
- `photo3.jpg` - Photo urbaine 3
- `photo4.jpg` - Photo urbaine 4
- `photo5.jpg` - Photo urbaine 5
- `photo6.jpg` - Photo urbaine 6

**Format recommandé :** 1200x900px (ratio 4:3), format JPG

### Sculptures (4 images)
- `sculpture1.jpg` - Fragment Urbain
- `sculpture2.jpg` - Géométrie Brute
- `sculpture3.jpg` - Monolithe
- `sculpture4.jpg` - Abstraction Urbaine

**Format recommandé :** 800x800px (carré), format JPG

## Conseils pour les images

- Utilisez des images haute qualité
- Optimisez les images pour le web (compression JPG à 80-85%)
- Style urbain : béton, architecture, textures industrielles
- Éclairage : contrastes marqués, ombres nettes
- Composition : géométrique, minimaliste

## Outils de compression recommandés

- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- ImageOptim (Mac)
- JPEGmini
